# 🚀 Portfólio Profissional - ADS

Projeto desenvolvido para a disciplina de **Fundamentos da Programação Web**.

## 🛠️ Tecnologias (Vanilla Stack)
Conforme exigido, o projeto não utiliza frameworks:
* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Layout responsivo com **Flexbox**.
* **JavaScript:** Acessibilidade (A+/A-) e validação de formulário.

## 📋 Seções Obrigatórias (Critérios de Avaliação)
1. **Sobre Mim:** Perfil e trajetória.
2. **Formação:** Histórico acadêmico.
3. **Portfólio:** Projetos e competências.
4. **Contato:** Formulário com validação nativa.
